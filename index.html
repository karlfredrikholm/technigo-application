What is a browser?

A web browser, or simply a 'browser' is an application used to access and view webpages.

When you type in the address of a webpage in a browser's address bar and press enter, a request is sent from your browser to the server where the webpage you want to visit is stored. The browser then retrieves and processes the data to make the requested webpage visible to you in the browser window.
